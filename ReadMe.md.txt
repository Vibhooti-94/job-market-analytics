# 📊 Job Market Analytics Dashboard

## 🚀 Project Overview

This project analyzes job market trends using a complete data pipeline from data generation to visualization.

The goal is to uncover insights related to job demand, hiring trends, salary distribution, and recruitment efficiency.

---

## 🛠️ Tools & Technologies

* Python (Pandas, Faker)
* SQL (PostgreSQL - Supabase)
* Power BI
* DAX

---

## 🔄 Data Pipeline

Supabase (PostgreSQL) → Google Colab → Pandas → CSV → Power BI Dashboard

---

## 📂 Dataset

The dataset includes:

* Job roles
* Companies
* Applications
* Salaries
* Candidate experience
* Hiring status

---

## 📊 Dashboard Features

* KPI Metrics (Total Applications, Hiring Rate, Average Salary)
* Job demand analysis
* Hiring funnel visualization
* Salary insights by role
* Location-based job trends

---

## 📈 Key Insights

* Data Analyst roles show high demand
* Majority applications remain in "Applied" stage
* Hiring rate indicates competitive market
* Salary varies significantly across roles
